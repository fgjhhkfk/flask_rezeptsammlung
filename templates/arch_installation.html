Installation von Arch Linux

Die Installation die hier beschrieben wird, richtet sich zum groessten Teil, bis auf einige kleinere Anpassungen, am Arch Wiki aus.

Den Installer hier ( LINK ) herunterladen, und mit >>> dd bs=4M if=path/to/archlinux.iso of=/dev/sdx status=progress oflag=sync auf einen Datentraeger schreiben. Den Rechner neu starten, und von dem zuvor beschrieben Datentraeger booten.

Tastaturlayout:

Standarmaessig wird das amerikanische Layout geladen. Im Grunde bin ich zufrieden damit, falls das aber mal geaendert werden soll, kann man mit

loadkeys de-latin1

entsprechend anpassen

Vor der Partitionierung der Festplatte, muss unterschieden werden ob per UEFI oder BIOS gebootet wurde. Das kann durch

>>> ls /sys/firmware/efi/efivars

gemacht werden. Wird das Verzeichnis ohne Fehlermeldung aufgelistet, kann davon ausgegangen werden, dass es sich bei dem System um ein UEFI System handelt.

Netzwerkverbindung

Waehrend der Installation muss eine Netzwerkverbindung bestehen. Prinzipiell sollte das auch mit W-Lan funktionieren, hat es aber bei mir auch nach einigen Versuchen nicht getan.
Daher also die Installation am Kabel (ist auch die nobrainer Methode). Kabel rein --> fertig; Aber nur wenn DHCP am Router konfiguriert ist. Falls die IP Adresse statisch sein soll, oder sich per W-Lan verbunden werden soll, gibt es dazu im WIKI auch Hilfe.
Ob das mit der Netzwerkverbindung auch geklappt hat, kann via Ping auf z.B. archlinux.org festgestellt werden.

Systemzeit einstellen

Um die Systemzeit einzustellen, muss erstmal die richtige Zeitzone verlinkt werden. Verfuegbare Zeitzonen koennen mit

>>> timedatectl list-timezones

angezeigt werden.
Mit

>>> ln -sf /usr/share/zoneinfo/Europe/Berlin /etc/localtime

wird dann auf die richtige Zeitzone verlinkt.
Mit

>>> timedatectl set-ntp true

wird dann die Netzwerksynchronisierung gestartet.
